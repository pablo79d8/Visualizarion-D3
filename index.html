Motivación

Este sitio web ha sido creado como parte de un proyecto académico para la asignatura Diseño y Evlaucación de Sistemas Interactivos de la Universidad de Valladolid.
El objetivo principal de este proyecto es la creación de una visualización que permita observar y analizar datos.
Los gráficos de líneas y mapas presentados aquí tienen un fin puramente educativo y no tienen ningún propósito comercial ni político.

Origen de los Datos

Los datos utilizados en este sitio web provienen del Instituto Nacional de Estadística (INE).
Estos datos son de acceso público y se obtienen a través del sitio web oficial del INE, y están disponibles para cualquier persona que desee utilizarlos.

Limitaciones de los Datos

Es importante tener en cuenta que los datos presentados aquí están sujetos a posibles actualizaciones.
Este sitio web no garantiza la exactitud de los datos, al no ser la información extraída de ellos su foco principal.

Responsabilidad del Usuario

Los usuarios de este sitio web deben comprender que cualquier interpretación, análisis o conclusiones extraídas de los datos presentados son responsabilidad exclusiva del usuario.
Este sitio web y sus creadores no se hacen responsables del uso inexacto de los datos.
